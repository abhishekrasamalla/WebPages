1)Simple selectors.
2)Combinator selectors
3)Pseudo -class selectors.
4)Pseudo -element selectors.
5)Attribute selectors.

	}
	</style>

CSS BOX MODEL: 

1) Content: This is the actual content of the element, such as text, images, or other media.

2) Padding: The padding is the space between the content and the element's border. It helps create space within the element.

3) Border: The border surrounds the padding and content. It can have a width, style, and color defined using CSS properties.

4) Margin: The margin is the space outside the border. It creates space between the element and other elements on the page.

CSS UNITS : 

1) px     (Absolute size)
2) %      (Related to parent width/height)
3) vw/vh  (Related to screen size)
4) rem    (Related to root element font)
5) em     (Related to its font size)
